# arcBlock-ios
### Description

这是arcblock code test项目

### Useage

1. open terminal
2. git clone https://github.com/TimberTang/arcBlock-ios.git
3. cd arcBlock
4. git checkout master
5. pod install (可以省略这一步， 防止cocoapod版本不一样)
6. open ArcBlock.workSpace
7. Select a iOS simulator 
8. run and review
